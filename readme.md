Projeto imaginado para criar um Organizador de Finanças Pessoal, utilizando Python e Django.

O projeto está dividido em 4 fases:

1 - Lógica do negócio e criação de banco de dados
2 - Criação de telas e formulários para as operações
3 - Implementação de planilhas e gráficos
4 - Estilização

A princípio o projeto tem duração estimada de 3 semanas.
